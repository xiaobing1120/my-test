# 思维导图 (Mind Map) 功能规格

## 1. 项目概述
- **项目名称**: Mind Map Canvas
- **项目类型**: React + Vite Web 应用
- **核心功能**: 基于 Canvas 的交互式思维导图，支持节点的编辑、拖动、层级重组和标签管理
- **目标用户**: 需要可视化思维整理的用户

## 2. UI/UX 规格

### 布局结构
- **全屏 Canvas**: 占满整个视口
- **工具栏**: 固定在顶部，包含添加节点、导出等操作按钮

### 视觉设计
- **背景色**: `#f5f7fa` (浅灰白)
- **主色调**: `#4a90d9` (蓝色)
- **节点背景**: `#ffffff`
- **节点边框**: `#d0d7de`
- **节点选中边框**: `#4a90d9`
- **连接线颜色**: `#8c959f`
- **标签预设颜色**:
  - 红色: `#ff6b6b`
  - 橙色: `#ffa94d`
  - 黄色: `#ffd43b`
  - 绿色: `#69db7c`
  - 蓝色: `#74c0fc`
  - 紫色: `#b197fc`
  - 粉色: `#f783ac`

### 节点设计
- **节点形状**: 圆角矩形 (border-radius: 8px)
- **节点阴影**: `0 2px 8px rgba(0,0,0,0.1)`
- **节点最小尺寸**: 宽 120px, 高 40px
- **节点内边距**: 12px 16px
- **文字字号**: 14px
- **标签样式**: 圆角小标签 (border-radius: 4px), 内边距 4px 8px, 字号 12px

### 交互状态
- **默认状态**: 正常显示
- **悬停状态**: 轻微放大 (scale: 1.02)
- **选中状态**: 蓝色边框 (2px)
- **拖动状态**: 透明度 0.8, 跟随鼠标
- **编辑状态**: 显示输入框

## 3. 功能规格

### 3.1 节点操作
- **添加根节点**: 点击工具栏按钮创建根节点
- **添加子节点**: 选中节点后点击添加按钮或按 Tab 键
- **删除节点**: 选中节点后按 Delete 键或点击删除按钮
- **编辑节点**: 双击节点进入编辑模式，直接在节点内输入文字

### 3.2 拖动功能
- **节点拖动**: 拖动节点移动位置
- **层级重组**: 拖动节点到另一个节点上，成为其子节点
- **拖动预览**: 拖动时显示半透明预览

### 3.3 标签功能
- **添加标签**: 选中节点后，点击标签按钮添加标签
- **编辑标签**: 点击标签进入编辑模式
- **删除标签**: 选中标签后按 Delete 键
- **选择颜色**: 弹出颜色选择器

### 3.4 画布操作
- **平移画布**: 拖动空白区域
- **缩放画布**: 鼠标滚轮缩放 (0.5x - 2x)

### 3.5 数据持久化
- **自动保存**: 数据保存在 localStorage
